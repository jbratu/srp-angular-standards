Control Error Message

This shows a custom error message in a validator by:

  1. The directive appControlErrorMessage listens to validation changes on the control.
  2. The validator validatorCustomMessage performs the validation on the control and in the process returns an error with the reason why validation failed.
  3. The appControlErrorMessage receives the event and appends HTML after the control to show the error.
  4. When appControlErrorMessage receives a valid status it clears the HTML by finding it with a CSS query selector.
